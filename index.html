adefault.css

Try out default, light and dark themes. These lightweight themes are intended to use as a basis of your own style sheets. Default theme contains Eric Meyer's reset and minimal set of typography styles. Light and dark themes contains basic color schemas for text, tables and form controls. No classes used, only tag styles.

  • adefault.css (3.7Kb) adefault.min.css (2.7Kb)
  • adefault-light.min.css (4.0Kb, including default)
  • adefault-dark.min.css (4.0Kb, including default)

Source code at GitHub contains also command-line theme generator (lessc needed to run it).
